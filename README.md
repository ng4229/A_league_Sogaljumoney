## 2020 미니드론 자율비행 경진대회
### [✔ 대회 진행 전략] 
* 여러가지 상황에 대한 변수를 가장 최우선적으로 고려함.
* HSV값으로 색 인식
* 홀을 완벽하게 인식하는 것으로 기본으로 하여 링 통과
* 침식과 팽창을 사용한 노이즈 제거
* 링 중앙점 인식 정확도를 높임
### [✔ 알고리즘 설명]

2. 링의 중앙점에 드론이 - 위치 해 있는지- 인식

  + 상하좌우 조정

* 드론이 링의 중앙점에 위치하지 않고 최적 거리 밖에 있다면 전진하여 드론이 링의 중앙점에 위치하고 최적 거리로 오도록 유도
